Renderjs in three lines

renderjs.createRenderer(canvas).then(renderer => {
    renderer.addNode(renderjs.Primitives.cube());
    renderer.start();
});

The slightly longer story

Renderjs is a framework for building 3D web applications. The principles behind its design are:

  • Simplicity
  • Non-intrusiveness
  • Flexibility

The rendering engine takes a scene graph and continuously renders it using WebGL. The beauty is in the design of the scene graph: it's a JSON-friendly format that uses expressions and bindings to create a dynamic scene from static data.

The upshot of this is that it's very easy to create a simple 3D scene, and at the same time you don't lose out on the power and flexibility of modern graphics cards. For example, you can define your own shaders on a per-object basis:

{
    children: [ {
        id: 'standard-render',
        mesh: { ... }
    }, {
        id: 'custom-render',
        mesh: { ... },
        shaders: {
            fragment: 'shaders/my-custom-shader.es2'
        }
    }]
}
